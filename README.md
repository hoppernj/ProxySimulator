# SpotProxy Simulation Modules

This repository includes core modules for simulating different proxy and censorship strategies.

---

## Modules

### 1. **SimProxy**
Simulates various proxy strategies using a simplified and optimized structure.  
Supported modes:
- Distributor: `strict`, `kind`, `snowflake`, `antizag`, `rbridge`
- Censor: `greedy`, `optimal`, `aggressive`, `zigzag`, `conservative`, `profile`, `targeted`
- Environment: `dynamic`, `ephemeral`, `collateral`
- Multiple or single-censor deployments-
